Поддерживаемые форматы

Word и его друзья

  1. doc ( результат, результат в html )
  2. docx ( результат, результат в html )
  3. odt ( результат, результат в html )

Разные формы сканированных документов

  1. pdf (с текстовым слоем и без) ( результат, результат в html )
  2. djvu или djv ( результат, результат в html )
  3. jpg ( результат, результат в html )
  4. png ( результат результат в html )
  5. Мультистраничные таблицы ( результат, результат в html )

Excel и его друзья

  1. ods (openoffice excel) ( результат результат в html )
  2. xls (старый excel) ( результат, результат в html )
  3. xlsx (новый excel) ( результат, результат в html )

CSV comma separated values
