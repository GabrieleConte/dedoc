How to parse documents

Upload the file here and look at the result. 

Build and Run

sudo docker-compose build doc_reader
sudo docker-compose up doc_reader
-> Service should rise at port 1231

Service parameters are configured in the config file (docreader/config.py)

The config is a python file, so you can use everything that standard python can do, for example, calculate the maximum file size as 512 * 1024 * 1024

How to use

You must send the file using the POST request to the address host:1231/upload

The name of the downloaded file should appear on the form

Additional query options:

  1. language: string - document recognition language. The default value is "rus+eng". Available values: "rus+eng", "rus", "eng".
  2. with_attachments: boolean - option including analysis of attached files. The option is False by default. Available values: True, False.
  3. document_type: string - document type. The service contains classifiers that extract the structure in different ways depending on the type of document. The difference lies in the use of different types of nodes and rules for parsing the structure of the document into the final tree. This option does not change the output format of parsing a document. While there are only 2 types of documents: "pdf" - ordinary documents, "law" - laws.
